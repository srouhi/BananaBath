# BananaBath – Interior Design Assistant  
*Hackathon Project – CornHacks 2025*

BananaBath is a full-stack interior design assistant specialized in bathroom design, developed during a 24-hour hackathon. The project combines a responsive frontend with an intelligent backend to provide design suggestions, color analysis, and image retrieval based on user input.

## Features
- **Chatbot Assistant:** Uses Gemini API and prompt engineering to provide cost analysis, color matching, and design recommendations.  
- **Mini RAG System:** Employs Semantic Transformer models and Cosine Similarity to retrieve relevant design images from a clean, manually curated dataset.  
- **Interactive Frontend:** Built with HTML, CSS, and JavaScript, providing a user-friendly interface for seamless interaction.  
- **Backend Integration:** Flask connects frontend and backend to create an end-to-end full-stack application.

## Tech Stack
- **Backend:** Python, Flask, Semantic Transformers, Cosine Similarity  
- **Frontend:** HTML, CSS, JavaScript  
- **APIs:** Gemini API  
- **Other:** Prompt engineering for chatbot responses
